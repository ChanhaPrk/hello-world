#깃허브를 사용하는 아주 간단한 방법
0. git clone url 를 이용하여 깃허브에 있는 파일들을 카피해온다.
1. git init를 사용하여 local git 과 git hub를 연결한다.
2. git status를 사용하여 현재 나의 상태를 살펴본다.
3. git add . 작업한 모든 파일을 워크스테이션에 올린다.
4. git commit -m "commit message"를 사용하여 커밋을 수행하며, -m를 이용하여 커밋코멘트를 남긴다.
5. git push origin master  master브렌치에 commit한 파일들을 업로드한다.
	- "psuh 하기전에 pull이 존재한다"를 잊지말자.
